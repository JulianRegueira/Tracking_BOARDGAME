# Tracker de Comandante - Versión Individual

App web/PWA simple para que cada jugador lleve en su propio celular:

- Nombre del jugador
- Vida
- Ataque
- Defensa

No necesita servidor ni base de datos. Guarda el estado localmente en el celular usando localStorage.

## Cómo publicarla en GitHub Pages

1. Subí estos archivos a la raíz de tu repositorio.
2. En GitHub, andá a Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /root.
6. Guardá.

La app debería quedar disponible en:

https://TU_USUARIO.github.io/NOMBRE_DEL_REPO/
